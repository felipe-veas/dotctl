# V1 Delivery Milestones

## Initial staged plan

- M0: bootstrap CLI and local config.
- M1: manifest resolution and apply engine.
- M2: Git integration and auth hardening.
- M3: tray apps for macOS and Linux.
- M4: reliability and security hardening.

## Success criteria theme

- Cross-platform behavior parity.
- Idempotent sync lifecycle.
- Actionable diagnostics and safe failure handling.
